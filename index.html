Aguarde — não feche a aba. Na 1ª vez o download dos pesos pode levar vários minutos (ex.: Gemma 2B ~1,5 GB em partes). Modelos 3B+ exigem mais VRAM; se falhar ao trocar, use F5 e comece por um modelo Leve. A barra pode ficar em 0% por alguns segundos antes de atualizar.

O modelo roda no seu navegador via WebGPU. Depois do cache, trocar de modelo fica bem mais rápido. 


      

    


    

      

        
Como funciona

        

          
1. WebLLM

          

            WebLLM Demo
            executa um LLM inteiro no browser com WebAssembly + WebGPU — sem API key e sem servidor de inferência.
          

        

        

          
2. Web Worker

          
Download e geração rodam em um Worker separado para a interface não travar.
